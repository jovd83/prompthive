# Language Selector Wireframe Specification

## Overview
Added a language selection mechanism to the application to support English (default), Dutch, and French.

## Before
- **Sidebar**: No language option.
- **Settings Page**: Account/Backup settings only.

## After
## After
1.  **Settings Page**:
    -   Field: "General Settings"
    -   Sub-field: "Display Language"
    -   Control: Dropdown (Select) with supported languages.
    -   Action: Updates application language immediately.

## Design Justification
-   Placing it in the User Menu allows quick access without navigating away from the current work.
-   Adding it to the specific Settings page provides a standard location consistent with other applications.

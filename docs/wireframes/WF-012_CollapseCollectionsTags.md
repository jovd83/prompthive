# WF-012: Collapsible Collections and Tags

## Overview
Added collapse/expand chevron icons to "Collections" and "Tags" headers in the sidebar.

## Wireframe

### Expanded State (Default)
```
[Sidebar]
...
COLLECTIONS [v] (+) (...)
- Collection A
- Collection B
  - Subcollection

TAGS [v] (...)
#Tag1 #Tag2
...
```

### Collapsed State
```
[Sidebar]
...
COLLECTIONS [>] (+) (...)
(Hidden List)

TAGS [>] (...)
(Hidden List)
...
```

**Legend:**
- `[v]`: Chevron Down (Expanded)
- `[>]`: Chevron Right (Collapsed)
- Unchanged elements: `(+)` (Add), `(...)` (Menu)
